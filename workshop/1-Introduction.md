# Course Introduction

Hi, I'm Adam, and welcome to my TanStack Start Workshop!

The repo for this course is the top pin at https://github.com/arackaf - fm-tanstack-workshop.

Course requirements:

- Docker.

That's about it, other than Node and npm like you already have. I'm really, really sorry to require Docker, but I did want to use a real database, Postgres in this case, and Docker is just the simplest way to get that running.

Running the project

- clone the repo
- npm i
- Run `npm run pg-raw`
- Open a NEW terminal and run `npm run dev`
- Browse to localhost:3000
